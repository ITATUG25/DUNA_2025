# DUNA_2025
Repositório de memorial descritivo de Cálculo da embarcação
O presente memorial descritivo tem como objetivo apresentar o desenvolvimento do projeto da embarcação VANIR, um rebocador em escala reduzida criado pela equipe ITATUG, composta por estudantes dos cursos de engenharia do CEFET/RJ – campus Itaguaí, para participação no Desafio Universitário de Nautidesign (DUNA). Esta competição nacional propõe a modelagem, construção e testes de um rebocador em escala, promovendo a aplicação prática de conhecimentos multidisciplinares das áreas de engenharia naval, mecânica, elétrica e gestão de projetos.
Neste primeiro ano de atuação construindo seu primeiro rebocador chamado Vanir, a equipe pôde adquirir diferentes conhecimentos extracurriculares incentivados pelo DUNA. Com isso, este documento apresenta o memorial de cálculo que fundamentam as decisões técnicas do projeto e a metodologia utilizada para a fabricação da embarcação.
